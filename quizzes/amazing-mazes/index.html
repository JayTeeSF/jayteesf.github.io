Amazing Mazes

Generate a rectangular perfect maze: every cell is reachable and exactly one simple path connects any pair of cells. Render it in a deterministic text form for a supplied seed.

graphsgenerationvisualization

Your task

  • Honor runtime width and height.
  • Visit every cell.
  • Create no cycles.
  • Produce the same maze for the same seed.

Example shape

width=8, height=5, seed=42 -> deterministic ASCII maze

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Randomized depth-first search creates a spanning tree.
  • Randomized Kruskal builds passages by joining disjoint sets.
  • Recursive division starts with open space and adds walls.
