2b re-scoped: the launcher boxed-arc campaign (spec 950 flip)

Fable-authored after the 2b honest-non-flip (2026-07-23). The slice-2

census ("~20 facades") was wrong: the dominant work is structural

boxed-collection lowering across tools/hey_selfc.hey and

tools/hey_run.hey (30+ functions each, the two largest Hey programs).

The non-flip run pinned 16 blocker classes (list in the 2b report,

preserved below) and proved one hard rule with a live regression:

RULE: any handler touching a SHARED evaluator (str_expr_value,

boxed_path_value, boxed_concat_value, cmp_expr_value, the generic

fac_args loop) gets its own corpus-wide survey IN THE SAME EDIT CYCLE

— the Files.join str-handler intercepted hey_semantics_tooling's

existing in-class `ret str Files.join(...)` and flipped the survey to

99/1 fail-closed. Additive dedicated-symbol handlers are exempt from

per-handler surveys (new prefixes cannot intercept existing routing)

but still ride the slice-final survey.

OPERATIONAL: both 950 fixtures SEGFAULT emit-hir at the default 8 MB

stack — `ulimit -s 65520` before interpreted-pipeline runs on them

(record also kept in lessons.md).

Slices (one gated commit each, alpha per slice)

  • 2b-i (safest, first): additive dedicated boxed facades pinned in the

non-flip run — Crypto.sha256_file, Files.metadata/ensure_dir/

remove_tree, Env 0-arg generic special-case (`ptr null, i64 0`;

split_elements('') spurious-arg guard), Crypto.sha256_text dedicated

no-clone str-return + generic-in-Text.slice context split.

  • 2b-ii: boxed index/comparison widening in boxed_path_value +

cmp_expr_value (arithmetic index, string-key bracket,

boxed-vs-boxed and boxed-vs-str-local strcmp), spread-array store

with boxed-index element. PER-HANDLER SURVEYS.

  • 2b-iii: corpus-risky str-context handlers (Files.join/dirname

chains, Text.starts_with? concat-arg leaf, Env.default nested

select) — these shadow existing generic routing; survey per handler,

plus targeted fixtures for the shapes they must NOT intercept.

  • 2b-iv: Version.current() const-fold, threaded once (interpolation +

str-concat + boxed-concat share it); HeyLlvm may import Version

(confirmed compiler-library-safe).

  • 2b-v: the unreached tail — Process.run_env_argv/run_argv/

exec_env_argv (boxed env-record + argv-array args), Files.

write_atomic/move/temp_dir, Json.decode/encode_canonical_line,

HeyParser.parse_file_cached, record spread {...extra, ...}, and

.ok/.stdout/.stderr/.value field access on facade results. Expect

this to split further after 2b-ii's structural work lands.

  • 2b-final: both fixtures byte-identical; flip 950's two pins; parity

loop +2; spec 950 solo (serial). Rung 2 route work completes here.
