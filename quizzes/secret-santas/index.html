Secret Santas

Assign every participant exactly one recipient while preventing self-assignment and same-family assignment. Produce a complete one-to-one matching or report that the constraints are impossible.

backtrackingconstraintsmatching

Your task

  • Each giver appears once.
  • Each recipient appears once.
  • No giver receives their own name.
  • No assignment stays within the same family.
  • Return an explicit error when no valid matching exists.

Example shape

Ada Lovelace -> Grace Hopper
Grace Hopper -> Alan Turing
Alan Turing -> Ada Lovelace

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Shuffle-and-retry is simple but nondeterministic and can loop.
  • Backtracking with early constraint checks is deterministic and testable.
  • Bipartite matching scales better when the rule set grows.
