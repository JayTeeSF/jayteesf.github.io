v0.68: Bootstrap handoff gate

v0.68 is **not** the fully self-hosted compiler.

It adds a Hey-written bootstrap gate report that checks the active compiler

migration slices:

  • source manifest/AST metadata,
  • simple-subset parser,
  • tiny Hey-to-C emitter,
  • self-compile pilot.

The report intentionally says `full_self_hosted: false` and points at `v0.72` as

the full self-hosting target. Native `heyc` remains the trusted oracle until the

self-hosted compiler path passes the full spec/example suite.
