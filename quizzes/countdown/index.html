Countdown Numbers

Given six numbers and a target, combine some or all inputs with addition, subtraction, multiplication, and exact division. Each input may be used at most once. Find an exact expression or the closest reachable value.

searchpruningarithmetic

Your task

  • Use each selected input no more than once.
  • Allow only positive intermediate values.
  • Allow division only when it is exact.
  • Prefer exact results; otherwise report the smallest absolute error.

Example shape

numbers: 75, 2, 8, 5, 10, 10
target: 926
result: an expression and its evaluated value

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Recursively choose unordered pairs and replace them with each legal result.
  • Memoize canonical multisets to avoid exploring equivalent states.
  • Use branch-and-bound once a near-target result is known.
