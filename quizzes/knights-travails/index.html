Knight's Travails

Find a shortest legal path for a chess knight between two squares while avoiding forbidden squares. Return the path, or no result when the destination cannot be reached.

graphsbreadth-first searchpaths

Your task

  • Use ordinary knight moves.
  • Stay on the 8x8 board.
  • Never land on a forbidden square.
  • Return a shortest path, including start and destination.

Example shape

start: a8
finish: b7
forbidden: b6
result: one shortest legal path

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Breadth-first search guarantees a shortest unweighted path.
  • Store predecessor links instead of copying every partial path.
  • A bidirectional search reduces work for larger generalized boards.
