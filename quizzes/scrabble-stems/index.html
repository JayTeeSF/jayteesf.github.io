Scrabble Stems

Find six-letter stems that can accept many different seventh letters and rearrange into valid seven-letter dictionary words. Rank stems by the number of distinct extension letters.

anagramsindexestext

Your task

  • Treat anagrams as the same signature.
  • Count distinct extension letters, not total words.
  • Accept a minimum cutoff.
  • Return deterministic ranking and tie-breaking.

Example shape

dictionary words -> stem signature -> distinct extension letters -> matching words

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • For every seven-letter word, remove each position and sort the remaining six letters.
  • Index words by sorted signature before deriving stems.
  • Use byte-frequency signatures to avoid repeated string sorting.
