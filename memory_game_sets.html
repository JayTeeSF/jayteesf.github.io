Printable Memory Game Sets

Each set below has a front PDF (the card faces) and a matching back PDF (the card backs with the set name). Print the fronts, then re-feed the same sheets to print the backs on the other side. Cut along the grid lines to create your deck.

How to Play

Most sets include a rules card, but the core idea is simple: shuffle the nine cards, place them face-down in a 3×3 grid, then take turns trying to reveal the entire sequence in the correct order. Players may either explore by flipping a single new card, or attempt to run the full sequence in one turn. If a card is flipped out of order, the turn ends and all cards are turned face-down again.
