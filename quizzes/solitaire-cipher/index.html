Solitaire Cipher

Implement the deck-based Solitaire hand cipher. Normalize text, generate a keystream by repeatedly transforming a 54-card deck, and use the stream to encrypt or decrypt letters.

arraysstate machinescryptography

Your task

  • Represent the two jokers distinctly.
  • Apply joker moves, the triple cut, and the count cut in the documented order.
  • Skip joker-valued output cards.
  • Support both encryption and decryption with deterministic test decks.

Example shape

plaintext:  CODE IN RUBY LIVELONG
normalized: CODEINRUBYLIVELONG
operation: encrypt, then decrypt back to the normalized text

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Model every deck transformation as a pure function returning a new deck.
  • Use a small state object containing the deck and emitted keystream.
  • For performance experiments, compare immutable rebuilding with an actor-owned mutable deck capability.
