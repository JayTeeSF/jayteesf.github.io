LCD Numbers

Render decimal digits as scalable seven-segment LCD characters. The requested size controls horizontal segment width and vertical segment height.

renderingstringstables

Your task

  • Support digits 0 through 9.
  • Use a default size when none is supplied.
  • Reject non-positive sizes.
  • Keep exactly one column of space between rendered digits.

Example shape

input: size=2, digits=012
output: a multi-line seven-segment display

Approach ideas

These are strategy summaries, not copied Ruby solutions.

  • Represent each digit as a seven-bit segment mask.
  • Use three logical rows, expanding vertical rows by size.
  • Precompute glyph rows for repeated rendering.
