Multiline strings

Hey provides two multiline string forms.

Use triple double quotes when interpolation is wanted:

let name = 'Ada'
let page = """<h1>Hello #{name}</h1>
<p>Welcome to Hey.</p>
"""

Use triple single quotes for raw multiline text. Raw multiline strings preserve

`#{...}` literally instead of evaluating it:

let template = '''Hello #{name}
The interpolation marker above remains text.
'''

Both forms preserve embedded newlines. Their delimiters are not part of the

resulting value. Ordinary single-quoted strings remain the concise raw form for

one-line text.
