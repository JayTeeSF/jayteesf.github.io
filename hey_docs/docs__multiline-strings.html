Multiline strings

Hey supports triple-quoted strings for readable help text, templates, and documentation snippets.

Use `"""` when a string naturally spans lines:

fn usage()
  return """Usage:
  ./bin/hey-vendor --repo PATH [options]
  ./bin/hey-vendor --vendor-dir PATH [options]
"""
end

The opening and closing `"""` delimiters are not part of the value. Newlines inside the block are preserved as `\n` characters in the resulting string.

Triple-quoted strings are equivalent to ordinary double-quoted strings after parsing, so interpolation works:

let name = 'Ada'
let page = """<h1>Hello #{name}</h1>
<p>Welcome to Hey.</p>
"""

Use single-quoted strings for short raw strings, double-quoted strings for ordinary one-line strings, and triple-quoted strings for multi-line text.
