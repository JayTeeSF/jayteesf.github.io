    Retirement Scenario Lab

    A single-file planning lab inspired by the RightCapital tax-strategy workflow. Model Roth conversions, RMD pressure, widow’s trap risk, relocation trade-offs across California, Texas, and Georgia, plus optional charitable / DAF deduction planning and post-retirement withdrawal-transfer tests using one household snapshot.

    Household

    Core demographic assumptions and spending targets.

    Strategy controls

    Roth conversion rules, return assumptions, and detail view. Earned income is modeled separately below, with optional charitable and transfer planning further down.

    Accounts

    Enter a current snapshot. Transactions are optional unless you want deeper spending calibration.

    Account name Owner Type Balance Cost basis Growth / rate Dividend yield Term yrs Annual pay Extra princ. Notes
    Import format is simple JSON. This template now supports checking, taxable, ira, roth, and mortgage. When you use mortgage schedules, keep annualSpending focused on non-mortgage living expenses so the scheduled mortgage payment is not double counted. If scheduledAnnualPayment is left at 0, the model now auto-computes an amortizing annual payment from the current balance, rate, and remaining term.

    Income sources

    Active pre-retirement earnings. Reuse the same source ID across multiple rows to split one paycheck across several target accounts without double-counting gross income.

    Source ID Name Owner Start year End year Annual gross Growth Gross tax Flow Contribution Value / target Priority Target account Notes
    Rows sharing one source ID are treated as one paycheck. Use Pretax to account for 401(k)-style payroll deferrals, After-tax to account for post-tax savings or mortgage paydown, and Expense for salary-linked cash outflows. After-tax rows are now funded from surplus cash only, in priority order, so the model no longer sells taxable assets just to force a savings contribution. Use Fill target balance with checking or mortgage targets to express rules like “top checking up to $100k, then send overflow elsewhere.” If endYear is 0, income stops when that owner reaches retirement age. For conservative estimates, set each growth rate to 0 or use the Zero income growth button. Use the dedicated charitable / DAF plans below when you want gifts to show up as both cash outflow and tax deduction.

    Charitable / DAF plans

    Optional annual gifts that increase cash needs and can offset taxes when itemizing beats the standard deduction.

    Enabled Name Owner Start year End year Mode Value Deduction mode Notes
    Modes: fixed dollars, percent of projected ordinary income, or percent of projected gross inflows. This is a planning approximation: the model only compares charitable deductions against the standard deduction; it does not model detailed Schedule A limitations.

    Post-retirement withdrawal-transfer plans

    Optional post-retirement tests that pull from IRA or cash, reserve estimated tax if desired, and redeploy the remainder into a new growth bucket.

    Enabled Name Owner Start year End year Annual gross Growth Source Destination Destination type Deposit mode Tax reserve rate Notes
    Net-after-estimated-tax deposits only the after-tax remainder into the destination account and parks the estimated tax reserve in checking so the later tax bill can consume it. A Roth destination is modeled as a Roth-like tax-free bucket for scenario testing; IRS contribution rules are not validated here.
