Feature Dependency Map

specs -> compiler tests -> generated C -> native executable
runtime values -> JSON/hash objects -> JSON arrays -> actor messages
functions -> modules -> module loading -> shippable programs
actors -> actor state -> process ring -> supervisors
errors -> failure specs -> supervisor crash policies
docs/tests -> release scripts -> downloadable binaries

Current priority

1. Supervisor runtime.

2. Module loading/multiple source files.

3. Shippable no-Ruby binary prototype.

4. Cross-platform release pipeline.

Deliberately deferred

• Classes.

• Inheritance.

• User-facing async/await.

• Full macro system.

• LLVM backend before runtime semantics settle.
